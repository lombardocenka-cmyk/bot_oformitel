# Как использовать Python 3.11 вместо 3.14

## ✅ Зависимости установлены!

Все необходимые пакеты успешно установлены для Python 3.11.1.

## Способы использования Python 3.11

### Способ 1: Использовать `py -3.11` (рекомендуется)

Для запуска бота:
```powershell
py -3.11 main.py
```

Для установки новых пакетов:
```powershell
py -3.11 -m pip install название_пакета
```

### Способ 2: Использовать готовый скрипт

Я создал файл `run_bot.bat` - просто дважды кликните на него, и бот запустится с Python 3.11.

### Способ 3: Сделать Python 3.11 по умолчанию

Если хотите, чтобы `python` всегда использовал версию 3.11:

1. **Найдите путь к Python 3.11:**
   ```
   C:\Users\Lombard\AppData\Local\Programs\Python\Python311\
   ```

2. **Добавьте в PATH:**
   - Нажмите `Win + R`, введите `sysdm.cpl` и нажмите Enter
   - Перейдите на вкладку "Дополнительно"
   - Нажмите "Переменные среды"
   - В "Системные переменные" найдите `Path` и нажмите "Изменить"
   - Нажмите "Создать" и добавьте:
     ```
     C:\Users\Lombard\AppData\Local\Programs\Python\Python311\
     C:\Users\Lombard\AppData\Local\Programs\Python\Python311\Scripts\
     ```
   - **ВАЖНО:** Переместите эти пути ВВЕРХ списка (выше пути к Python 3.14)
   - Нажмите "ОК" везде
   - **Перезапустите PowerShell/терминал**

3. **Проверьте:**
   ```powershell
   python --version
   ```
   Должно вывести: `Python 3.11.1`

### Способ 4: Создать виртуальное окружение с Python 3.11

```powershell
# Создайте виртуальное окружение с Python 3.11
py -3.11 -m venv venv311

# Активируйте его
.\venv311\Scripts\Activate.ps1

# Если получите ошибку о политике выполнения:
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

# Теперь python будет использовать версию 3.11
python --version
python main.py
```

## Проверка установки

Проверьте, что все работает:

```powershell
py -3.11 -c "import aiogram; print('✅ aiogram установлен!')"
py -3.11 -c "import aiohttp; print('✅ aiohttp установлен!')"
py -3.11 -c "import aiosqlite; print('✅ aiosqlite установлен!')"
```

## Запуск бота

```powershell
# Способ 1: Через py launcher
py -3.11 main.py

# Способ 2: Через скрипт
.\run_bot.bat

# Способ 3: Если настроили PATH
python main.py
```

## Важно!

- **Не используйте просто `python`** - он все еще указывает на Python 3.14
- **Используйте `py -3.11`** для всех команд Python
- Или настройте PATH, как описано выше

## Удаление Python 3.14 (опционально)

Если хотите полностью удалить Python 3.14:

1. Откройте "Параметры" → "Приложения"
2. Найдите "Python 3.14" или "Python Core"
3. Нажмите "Удалить"

⚠️ **Внимание:** Это удалит Python 3.14, но не обязательно - можно просто использовать `py -3.11`.
