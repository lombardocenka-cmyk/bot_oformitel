# Быстрая настройка бота

## ❌ Проблема: Токен бота не настроен

Файл `.env` существует, но в нем нет необходимых параметров.

## ✅ Решение: Заполните файл .env

### Шаг 1: Получите токен бота

1. Откройте Telegram
2. Найдите [@BotFather](https://t.me/BotFather)
3. Отправьте команду `/newbot`
4. Следуйте инструкциям:
   - Придумайте имя бота (например: "Мой магазин техники")
   - Придумайте username бота (например: `my_tech_store_bot`)
5. **Скопируйте токен** - он выглядит как: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`

### Шаг 2: Получите ваш Telegram ID

1. Откройте Telegram
2. Найдите [@userinfobot](https://t.me/userinfobot)
3. Отправьте команду `/start`
4. **Скопируйте ваш ID** - это число, например: `123456789`

### Шаг 3: Настройте канал

1. Создайте канал в Telegram (или используйте существующий)
2. Добавьте бота в канал как администратора:
   - Откройте настройки канала → "Администраторы"
   - "Добавить администратора" → найдите вашего бота
   - Дайте права на публикацию сообщений
3. Получите ID канала:
   - **Публичный канал:** используйте username (например: `@my_channel`)
   - **Приватный канал:** перешлите сообщение из канала боту [@userinfobot](https://t.me/userinfobot) - он покажет ID (формат: `-1001234567890`)

### Шаг 4: Заполните файл .env

Откройте файл `.env` в корне проекта и заполните его следующим образом:

```
BOT_TOKEN=123456789:ABCdefGHIjklMNOpqrsTUVwxyz
ADMIN_ID=123456789
CHANNEL_ID=@my_tech_channel
```

**ВАЖНО:**
- Замените `123456789:ABCdefGHIjklMNOpqrsTUVwxyz` на ваш реальный токен
- Замените `123456789` на ваш реальный Telegram ID
- Замените `@my_tech_channel` на ваш реальный канал

### Шаг 5: Проверьте настройки

Запустите скрипт проверки:

```powershell
py -3.11 check_config.py
```

Если все правильно, вы увидите:
```
✅ BOT_TOKEN найден: ...
✅ ADMIN_ID установлен: 123456789
✅ CHANNEL_ID установлен: @my_channel
✅ Все параметры настроены правильно!
```

### Шаг 6: Запустите бота

```powershell
py -3.11 main.py
```

## Пример заполненного .env

```
BOT_TOKEN=7123456789:AAHdqTcvCH1vGWJxfSeofSAs0K5PALDsaw
ADMIN_ID=123456789
CHANNEL_ID=@my_tech_store
```

## Помощь

Если возникли проблемы:
1. Убедитесь, что токен скопирован полностью (без пробелов)
2. Убедитесь, что ID администратора - это число
3. Убедитесь, что канал начинается с `@` или `-100`
4. Проверьте, что бот добавлен в канал как администратор

## Безопасность

⚠️ **НИКОГДА не публикуйте файл .env в интернете!**
- Добавьте `.env` в `.gitignore` (если используете Git)
- Не делитесь токеном бота с другими
