# Инструкция по настройке бота

## Шаг 1: Получение токена бота

1. Откройте Telegram и найдите [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/newbot`
3. Следуйте инструкциям и создайте бота
4. Скопируйте полученный токен (выглядит как `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

## Шаг 2: Получение вашего Telegram ID

1. Откройте Telegram и найдите [@userinfobot](https://t.me/userinfobot)
2. Отправьте команду `/start`
3. Бот пришлет ваш ID (число, например `123456789`)

## Шаг 3: Настройка канала

1. Создайте новый канал в Telegram или используйте существующий
2. Добавьте бота в канал как администратора:
   - Откройте настройки канала
   - Перейдите в "Администраторы"
   - Нажмите "Добавить администратора"
   - Найдите вашего бота и добавьте его
   - Дайте боту права на публикацию сообщений
3. Получите ID канала:
   - Если канал публичный (имеет username типа `@mychannel`), используйте его
   - Если канал приватный, перешлите любое сообщение из канала боту [@userinfobot](https://t.me/userinfobot) - он покажет ID канала (формат: `-1001234567890`)

## Шаг 4: Создание файла .env

Создайте файл `.env` в корне проекта со следующим содержимым:

```
BOT_TOKEN=ваш_токен_бота
ADMIN_ID=ваш_telegram_id
CHANNEL_ID=@ваш_канал или -1001234567890
```

**Пример:**
```
BOT_TOKEN=123456789:ABCdefGHIjklMNOpqrsTUVwxyz
ADMIN_ID=123456789
CHANNEL_ID=@my_tech_channel
```

## Шаг 5: Установка зависимостей

```bash
pip install -r requirements.txt
```

## Шаг 6: Запуск бота

```bash
python main.py
```

Если все настроено правильно, вы увидите сообщения:
```
INFO - Инициализация базы данных...
INFO - База данных инициализирована
INFO - Запуск планировщика постов...
INFO - Планировщик запущен
INFO - Бот запущен и готов к работе!
```

## Проверка работы

1. Откройте Telegram и найдите вашего бота
2. Отправьте команду `/start`
3. Бот должен приветствовать вас и предложить выбрать категорию товара

## Решение проблем

### Бот не отвечает
- Проверьте, что токен бота правильный в файле `.env`
- Убедитесь, что бот запущен (нет ошибок в консоли)

### Ошибка "Chat not found"
- Проверьте, что ID канала правильный
- Убедитесь, что бот добавлен в канал как администратор

### Ошибка "Forbidden: bot is not a member of the channel"
- Добавьте бота в канал как администратора с правами на публикацию

### Ошибка при поиске характеристик
- Это нормально, поиск характеристик работает упрощенно
- Можно вручную отредактировать характеристики в боте
